

## Instação



## Rodando o app

```bash
# development
$ npm run start

$ npm run start:dev

# production mode
$ npm run start:prod
```

```bash
# testes unitarios
$ npm run test

# e2e testes
$ npm run test:e2e

# teste coverage
$ npm run test:cov
```
