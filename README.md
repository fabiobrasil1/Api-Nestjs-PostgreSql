

## Instalação


```bash
$ npm install
```

## Rodando o app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Teste

```bash
# testes unitarios
$ npm run test

# e2e testes
$ npm run test:e2e

# teste coverage
$ npm run test:cov
```
